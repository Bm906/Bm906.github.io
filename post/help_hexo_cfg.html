注意事项

  • 步骤1、2，每台电脑需要安装一次

  • 步骤4，只用创建一次，可忽略

1. install Node

nodejs.org

推荐下载LTS（Long Time Support）版本

打开 Node.js command prompt

验证 version

2. install hexo

在Git Bash中安装 hexo

3. hexo 仓库

创建文件夹，D:\xx\hexo， 进入 hexo

4. Hexo deploy Github

创建github项目 Bm906.github.io

https://github.com/Bm906/Bm906.github.io

// 浏览器预览Hexo
https://Bm906.github.io

5. 换电脑，覆盖3个文件

这里用“hexo:”表示hexo根目录

6. 配置 _config.yml

hexo:_config.yml

  • 修改部署–deploy
  • 修改url–好像可不改
  • source–文档位置–不改
  • title

7. 配置 package.json

hexo:package.json

  • json依赖包–不改–换主题后可能要改
  • hexo版本–不改
